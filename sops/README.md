# Standard Operating Procedures

How the Mantrasoft AI-augmented development team works.

## SOPs

- [Session Workflow](session-workflow.md) — How a coding session picks up work, executes, and reports back
- [Commander Protocol](commander-protocol.md) — How the commander orchestrates the team
- [Issue Standards](issue-standards.md) — How to write issues, labels, root cause comments

## Principles

1. **Every task is a GitHub issue.** No work without an issue.
2. **Commander orchestrates, sessions execute.** Clear roles.
3. **Sessions self-orient.** `/session <project>` is all it takes to start.
4. **Everything is traceable.** Issue → commit → comment → close.
5. **One source of truth.** Atlas for structure, board for today, issues for tasks.
